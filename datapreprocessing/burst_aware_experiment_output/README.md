# Burst-Aware Experiment Outputs

This folder contains the focused experiment set for the adaptive temporal allocation method.

## Useful figures

1. `burst_canonical_patterns.png`
   - Synthetic behavior checks for uniform, single-burst, repeated-peak, and multi-burst cases.

2. `burst_real_windows_comparison.png`
   - Real Chicago crime windows shown under linear and adaptive allocation.

3. `burst_bin_count_sensitivity.png`
   - Sensitivity of the allocation to bin-count choice.

4. `burst_window_stability.png`
   - Stability of the allocation under small window shifts.

## Summary file

- `burst_experiment_summary.csv`
  - Stores the real windows selected for the comparison figure.

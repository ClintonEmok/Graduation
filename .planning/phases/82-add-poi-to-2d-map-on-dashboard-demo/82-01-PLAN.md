# 82-01-PLAN.md — Wire MapPoiLayer into demo map + add toggle

## Goal
Make POIs (police, schools, transit, parks) render on the dashboard-demo 2D map by default, with a "Show POIs / Hide POIs" toggle button at `top-4` above the STKDE button, and respect the existing `useDashboardDemoMapLayerStore` for visibility + opacity + persistence.

## Tasks

### Task 1: Render POI layer in `MapVisualization`
**File:** `src/components/map/MapVisualization.tsx`

- Import `MapPoiLayer` from `./MapPoiLayer`
- Inside the `<MapBase>` children, after `MapEventLayer` block, add:
  ```tsx
  {visibility.poi ? <MapPoiLayer visible categories={['police', 'schools', 'transit', 'parks']} /> : null}
  ```
- Wrap in a div with `style={{ opacity: opacity.poi }}` to apply layer opacity without changing the layer's API
  - Cleanest: a single wrapper div around the conditional layer:
    ```tsx
    <div className="absolute inset-0 pointer-events-none" style={{ opacity: opacity.poi }}>
      {visibility.poi ? <MapPoiLayer visible categories={...} /> : null}
    </div>
    ```
  - But MapPoiLayer renders react-map-gl `Marker` children which need to be inside `<Map>`. So instead apply the opacity via a CSS variable or use the layer's own wrapping div. The simplest correct approach: put the layer **inside** `<MapBase>` (as siblings of other layers) and apply opacity via a wrapper div that is a direct child of Map — this is valid because Marker accepts any container.

  **Final approach:** render the conditional inside `<MapBase>` but wrap in a `<div>` that sets `opacity` on its content. React-map-gl markers are positioned by `Marker`, but each marker is wrapped in a div that gets the opacity. Result: marker backgrounds + popups fade together.

### Task 2: Add POI toggle button in `DemoMapVisualization`
**File:** `src/components/dashboard-demo/DemoMapVisualization.tsx`

- Add `const poiVisible = useDashboardDemoMapLayerStore((state) => state.visibility.poi);` near line 50
- Add a third `Button` after the existing buttons, at `top-4` (before the STKDE button at `top-14`):
  ```tsx
  <Button
    type="button"
    onClick={() => toggleVisibility('poi')}
    aria-label={poiVisible ? 'Hide POI overlay' : 'Show POI overlay'}
    title={poiVisible ? 'Hide POI overlay' : 'Show POI overlay'}
    variant={poiVisible ? 'destructive' : 'outline'}
    size="sm"
    className="absolute right-4 top-4 z-40 gap-2 rounded-full text-[11px]"
  >
    <Layers3 className="size-3.5" />
    {poiVisible ? 'Hide POIs' : 'Show POIs'}
  </Button>
  ```

    3. `def compute_baseline(df: pd.DataFrame) -> pd.DataFrame:` — returns a 168-row DataFrame with columns `[hour, dow, count, mean_per_sec, sigma_per_sec, count_cell_weeks]`.
       - Group `df` by `(hour, dow)` and count events per cell → `count` (int64).
       - Compute the number of cell-weeks per (h, d) in the dataset: `ts_min, ts_max = df['ts'].min(), df['ts'].max(); total_seconds = ts_max - ts_min; total_weeks = total_seconds / (7 * SECONDS_PER_DAY); count_cell_weeks = total_weeks` (every (h, d) cell has the same count_cell_weeks = total weeks in the dataset, ~1330 weeks for 25 years).

Comment in the code: "Per-cell sigma uses a Poisson-style rate-based estimate. This is the null-model sigma — the actual σ would be computed from weekly bucket counts in Plan 02.          - The window spans some number of (h, d) cells. For each cell `(h, d)`, compute the fraction of the window that falls into that cell (use a simple bucketing approach: bin each event's ts into `(window_start // window_seconds)`, then count events per (h, d) per window).
         - Simpler: for each window, compute `observed = len(df_sorted[(df_sorted['ts'] >= start) & (df_sorted['ts'] < start + window_seconds)])`. Compute `expected = sum over (h, d) in window's cells of mean_per_sec[h, d] * seconds_in_cell[h, d]`. Compute `sigma = sqrt(sum over (h, d) in window's cells of sigma_per_sec[h, d]^2 * seconds_in_cell[h, d])`.
         - For the *quick preview*, use a fast approximation: since step = window/4, most windows will be entirely within one (h, d) cell.          - Compute `z = (observed - mu) / max(sigma, EPSILON)`.

    Module docstring: "Per-metric comparison summary (CBP-03). Reads the contextual and Goh-Barabasi per-window parquets, computes per-(metric, window_size) statistics, and writes a tidy long-form comparison_table.csv. The decision-gate in Plan 05 reads this CSV."

    1. `def summarize_metric(series: pd.DataFrame, value_col: str, metric_name: str) -> pd.DataFrame:`
       - Input: `series` is a per-window DataFrame (from a parquet), `value_col` is the column to summarise (e.g. "z" or "B"), `metric_name` is "contextual" or "goh_barabasi".
       - Output: a DataFrame with columns `[metric, window_size, window_label, n_windows, mean, std, min, max, cv, range]` and 4 rows (one per window size in WINDOW_LABELS).
       - Algorithm:
         a. Group by `window_sec` and apply population std (default in pandas/numpy — `ddof=0`).
         b. For each group: `n = len`, `mean = series[value_col].mean()`, `std = series[value_col].std(ddof=0)`, `min = series[value_col].min()`, `max = series[value_col].max()`, `cv = std / abs(mean) if mean != 0 else float("nan")`, `range = max - min`.
         c. Map `window_sec` to label via the `WINDOW_LABELS = {3600: "1h", 21600: "6h", 86400: "1d", 604800: "1w"}` constant (define at module level).
         d. Build the output DataFrame with 4 rows. Sort by `window_sec` ascending.

    2. `def build_comparison_table(contextual_df: pd.DataFrame, goh_barabasi_df: pd.DataFrame) -> pd.DataFrame:`
       - Calls `summarize_metric(contextual_df, "z", "contextual")` and `summarize_metric(goh_barabasi_df, "B", "goh_barabasi")`, then `pd.concat` them.
       - Output: 8-row DataFrame with the same columns.

    3. `def write_comparison_csv(table: pd.DataFrame, output_path: Path) -> None:`
       - `output_path.parent.mkdir(parents=True, exist_ok=True)`, `table.to_csv(output_path, index=False, float_format="%.6f")`.
       - Print `f"Wrote {len(table)} rows to {output_path}"`.

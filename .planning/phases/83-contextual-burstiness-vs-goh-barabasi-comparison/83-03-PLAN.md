    2. `def goh_barabasi_b(inter_event_gaps: np.ndarray) -> float | None:`
       - Direct port of `src/lib/burst-detection.ts:53-59`. The TS returns `0` for degenerate cases; **this port returns `None`** to match `83-RESEARCH.md` L107-119 design and let the caller drop degenerate windows cleanly.
       - Body (use exactly this):
         ```python
         if inter_event_gaps.size < 2:
             return None
         mean = float(inter_event_gaps.mean())
         if mean <= 0:
             return None
         variance = float(((inter_event_gaps - mean) ** 2).mean())
         std = math.sqrt(variance)
         denom = std + mean
         if denom == 0:
             return None
         return (std - mean) / denom
         ```
       - Add `import math` at the top.

    3. `def compute_goh_barabasi_series(timestamps: np.ndarray, window_sec: int, step_sec: int | None = None, min_events: int = MIN_EVENTS_PER_WINDOW) -> pd.DataFrame:`
       - `timestamps` shape `(N,)` int64 epoch seconds (sorted ascending).
       - `step_sec` defaults to `max(1, window_sec // 4)`.
       - Output: DataFrame with columns `[window_start, window_end, n_events, B]` and dtypes `[int64, int64, int32, float64]`.
       - Algorithm:
         a. If `len(timestamps) < 2`, return an empty DataFrame with the columns above.
         b. `t_start, t_end = int(timestamps[0]), int(timestamps[-1])`.
         c. For `ws in range(t_start, t_end - window_sec + 1, step_sec)`:
            - `we = ws + window_sec`.
            - `lo = np.searchsorted(timestamps, ws, side="left")`, `hi = np.searchsorted(timestamps, we, side="left")`.
            - `n_events = hi - lo`. If `n_events < min_events`, skip.
            - `gaps = np.diff(timestamps[lo:hi].astype(np.float64))`. If `gaps.size < 2`, skip.
            - `b = goh_barabasi_b(gaps)`. If `b is None`, skip.
            - Append `(ws, we, n_events, b)`.
         d. Build and return the DataFrame. If no windows survived, return an empty DataFrame (still has the columns).

    4. `def write_goh_barabasi_parquet(series: pd.DataFrame, output_path: Path) -> None:` — same shape as `write_contextual_parquet()`: mkdir parents, `to_parquet(index=False)`, print summary.

    5. `if __name__ == "__main__":` block:
       ```python
       from db import load_crimes
       from pathlib import Path
       import numpy as np
       df = load_crimes()
       timestamps = df["ts"].to_numpy()
       # Full-series IEI reference B (per docs/FUTURE-WORK-ADAPTIVE-TIME.md:288 ≈ 0.30)
       full_gaps = np.diff(timestamps.astype(np.float64))
       b_full = goh_barabasi_b(full_gaps)
       print(f"Full-series Goh-Barabasi B (IEI): {b_full:.4f}  (expected ≈ 0.30)")
       output_path = Path("output/goh_barabasi_metric.parquet")
       all_series = []
       for window_sec in WINDOWS_SEC:
           step = max(1, window_sec // 4)
           series = compute_goh_barabasi_series(timestamps, window_sec, step)
           series["window_sec"] = window_sec
           all_series.append(series)
           print(f"  {WINDOW_LABELS[window_sec]}: {len(series):,} valid windows")
       full = pd.concat(all_series, ignore_index=True) if all_series else pd.DataFrame()
       write_goh_barabasi_parquet(full, output_path)
       ```

    6. Imports: `from __future__ import annotations`, `numpy as np`, `pandas as pd`, `math`, `from pathlib import Path`.

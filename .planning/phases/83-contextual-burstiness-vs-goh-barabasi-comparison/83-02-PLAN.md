    Create `metrics/contextual.py` with two public functions and one helper, all fully type-hinted.

    Module docstring: "Contextual burstiness metric (CBP-01). Computes a 168-cell hour×dayOfWeek baseline (mu_c, sigma_c) on the full dataset, then a per-window z-score for each sliding window at 1h/6h/1d/1w with step=window/4. Mirrors the formulation in docs/FUTURE-WORK-ADAPTIVE-TIME.md L57-83 and the locked decision in 83-CONTEXT.md."

    1. Constants:
       - `WINDOWS_SEC = (3600, 21600, 86400, 604800)` — the locked 1h/6h/1d/1w set.
       - `WINDOW_LABELS = {3600: "1h", 21600: "6h", 86400: "1d", 604800: "1w"}` — matches `scripts/burstiness_sweep.py:204-210`.
       - `MIN_EVENTS_PER_WINDOW = 5` — matches `83-RESEARCH.md` Risk #7 floor for fairness with the Goh-Barabasi metric.
       - `SECONDS_PER_WEEK = 7 * 24 * 3600` = 604800.
       - `EPSILON = 1e-9` — for sigma=0 cell substitution per `83-RESEARCH.md` L88.

    2. `def compute_baseline(events: pd.DataFrame) -> tuple[np.ndarray, np.ndarray]:`
       - Input: `events` is a DataFrame with columns `ts:int64, hour:int8, dow:int8, month:int8` (as produced by `load_crimes()`).
       - Output: `(mu, sigma)` each shape `(24, 7)` dtype `float64`.
       - Algorithm:
         a. Compute `week_index = (events["ts"] // SECONDS_PER_WEEK).astype("int64")` — each timestamp maps to an integer week since 1970-01-01.
         b. `weekly = events.assign(week=week_index).groupby(["week", "hour", "dow"]).size().reset_index(name="count")` — for every (week, hour, dow) cell, the number of events that fell in that cell during that week.
         c. `per_cell = weekly.groupby(["hour", "dow"])["count"].agg(["mean", "std"])` — population std (default in pandas).
         d. Build a `(24, 7)` array for `mu` and `sigma`. Initialise with `np.zeros((24, 7), dtype=np.float64)`. For each `(h, d)` in `per_cell.index`, set `mu[h, d] = per_cell.loc[(h, d), "mean"]` and `sigma[h, d] = per_cell.loc[(h, d), "std"]` (NaN-safe — fill NaN with 0.0).
         e. Apply the `EPSILON` substitution: `sigma = np.where(sigma < EPSILON, EPSILON, sigma)`. Count the number of substituted cells and print to stderr: `f"Baseline: {n_substituted}/168 cells had sigma<{EPSILON}, substituted with epsilon"`.
         f. Return `(mu, sigma)`.

    3. `def _z_for_window(window_start: int, window_end: int, mu: np.ndarray, sigma: np.ndarray) -> tuple[int, int, int, float]:`
       - Linearised per-second form. Compute `dt = window_end - window_start` (float seconds). For every (h, d) in the 168 cells, compute the fraction of the week that the window covers in that cell (window-spans-at-most-2-cells is fine — handle the general case by iterating over the 168 cells and adding the contribution only if the window overlaps that cell).
       - Simpler and correct-enough implementation (use this — the per-cell overlap computation in `83-RESEARCH.md` L70-78 is more general but harder to read; for the locked 1h/6h/1d/1w sweep the windows align well enough with the (h, d) grid that the *contributing-cell-only* approximation is sufficient):
         a. The window covers some range of (h, d) cells. For each integer hour `h ∈ {0..23}` and day-of-week `d ∈ {0..6}` that the window *fully* covers, add `mu[h, d] * 3600 / SECONDS_PER_WEEK` to `expected` and `sigma[h, d]**2 * 3600 / SECONDS_PER_WEEK` to `sigma_W_sq`.
         b. For partial overlaps (the first and last hour of the window, if the window doesn't align on hour boundaries), compute the exact overlap seconds `Δt_cell` and add `mu[h, d] * Δt_cell / SECONDS_PER_WEEK` and `sigma[h, d]**2 * Δt_cell / SECONDS_PER_WEEK`.
         c. `expected = sum_mu_per_sec * dt` where `sum_mu_per_sec = sum of (mu[h,d] / SECONDS_PER_WEEK) over overlapping cells weighted by overlap seconds`.
         d. `z = (obs - expected) / sqrt(sigma_W_sq)`. Return `(window_start, window_end, primary_h, primary_d, z)` where `primary_h` and `primary_d` are the (h, d) of the window's midpoint.
       - Edge case: if `sigma_W_sq == 0`, return `z = float("nan")` (caller will filter).

    4. `def compute_contextual_z_series(timestamps: np.ndarray, hours: np.ndarray, dows: np.ndarray, mu: np.ndarray, sigma: np.ndarray, window_sec: int, step_sec: int | None = None) -> pd.DataFrame:`
       - Input: `timestamps` shape `(N,)` int64 epoch seconds (sorted ascending), `hours` and `dows` shape `(N,)` int8 (the hour/dow of each timestamp), precomputed baseline.
       - `step_sec` defaults to `max(1, window_sec // 4)` — matches `scripts/burstiness_sweep.py:282`.
       - Output: DataFrame with columns `[window_start, window_end, hour, dow, z, obs, expected, sigma_w]` and dtypes `[int64, int64, int8, int8, float64, int32, float64, float64]`. (`hour` and `dow` are the midpoint of the window, for indexing back into the heatmap in Plan 04.)
       - Algorithm:
         a. If `len(timestamps) < 2`, return an empty DataFrame with the columns above.
         b. `t_start, t_end = timestamps[0], timestamps[-1]`.
         c. For `ws in range(t_start, t_end - window_sec + 1, step_sec)`:
            - `we = ws + window_sec`.
            - Use `np.searchsorted` for slicing: `lo = np.searchsorted(timestamps, ws, side="left")`, `hi = np.searchsorted(timestamps, we, side="left")`.
            - `obs = hi - lo`. If `obs < MIN_EVENTS_PER_WINDOW`, skip.
            - Compute expected and sigma via `_z_for_window(ws, we, mu, sigma)`.
            - Append `(ws, we, primary_h, primary_d, z, obs, expected, sigma_w)`.
         d. Build and return the DataFrame.

    5. `def write_contextual_parquet(series: pd.DataFrame, output_path: Path) -> None:` — ensure `output_path.parent.mkdir(parents=True, exist_ok=True)`, then write `series.to_parquet(output_path, index=False)`. Print `f"Wrote {len(series):,} windows to {output_path}"` to stdout.

    6. `if __name__ == "__main__":` block:
       ```python
       from db import load_crimes
       from pathlib import Path
       df = load_crimes()
       print("Computing baseline (168 cells)...")
       mu, sigma = compute_baseline(df)
       print(f"Baseline mu range: [{mu.min():.2f}, {mu.max():.2f}]")
       print(f"Baseline sigma range: [{sigma.min():.4f}, {sigma.max():.4f}]")
       output_dir = Path(__file__).parent / "output"
       output_dir.mkdir(parents=True, exist_ok=True)
       output_path = output_dir / "contextual_metric.parquet"
       all_series = []
       for window_sec in WINDOWS_SEC:
           step = max(1, window_sec // 4)
           series = compute_contextual_z_series(df["ts"].to_numpy(), df["hour"].to_numpy(), df["dow"].to_numpy(), mu, sigma, window_sec, step)
           series["window_sec"] = window_sec
           all_series.append(series)
           print(f"  {WINDOW_LABELS[window_sec]}: {len(series):,} valid windows")
       full = pd.concat(all_series, ignore_index=True)
       write_contextual_parquet(full, output_path)
       ```

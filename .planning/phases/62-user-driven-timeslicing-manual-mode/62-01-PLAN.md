     * Set isCreatingManualSlice = true

   - onManualMouseMove: (e: MouseEvent) => void
     * If isCreatingManualSlice is true
     * Calculate current timestamp from mouse X
     * Update manualCreationEnd
     * Update ghostSliceStyle with:
       - position: absolute
       - left: scale(manualCreationStart)
       - width: scale(manualCreationEnd) - scale(manualCreationStart)
       - border: 2px dashed currentColor
       - background: rgba(currentColor, 0.2)
       - minimum width enforcement (200ms = minimum duration per D-07)
   
   - onManualMouseUp: (e: MouseEvent) => void

     * Reset isCreatingManualSlice, manualCreationStart, manualCreationEnd

     * If Escape pressed during creation, cancel creation
     * Reset all creation state

3. Add Ghost Slice Preview overlay:

   - Show tooltip with proposed duration and time range during drag

   - Extend existing handle drag logic or create new handlers

   - Drag updates slice boundaries via useBinningStore.resizeBin()
   - Snap-to-grid based on sliceSnapInterval from store

   - When mode === 'auto', disable creation (read-only timeline)

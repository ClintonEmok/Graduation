This plan introduces span-aware tick logic and route-scoped rollout controls so we improve UX on testing surfaces without changing dashboard behavior yet.
Output: Shared tick policy helpers, DualTimeline integration, audited route callsites, regression guards, and a route-only verification checkpoint.
</objective>

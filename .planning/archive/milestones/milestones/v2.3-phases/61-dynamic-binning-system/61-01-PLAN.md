<task type="auto">
  <name>Task 2: Implement preset binning rules</name>
  <files>src/lib/binning/rules.ts</files>
  <action>
Implement 12 preset strategies:
- daytime-heavy: Focus thresholds on 6am-6pm, detect day patterns
- nighttime-heavy: Focus thresholds on 6pm-6am, detect night patterns
- crime-type-specific: Group by crime type clusters using IUCR codes
- burstiness: Split by inter-arrival time distributions
- uniform-distribution: Equal events per bin using quantile binning
- uniform-time: Equal time span bins
- weekday-weekend: Separate weekday from weekend bins
- quarter-hourly: 15-minute fixed intervals
- hourly: 1-hour fixed intervals
- daily: 24-hour fixed intervals
- weekly: 7-day fixed intervals
- auto-adaptive: Auto-detect best strategy based on data distribution
  </action>
  <verify>Each strategy exports a generateThresholds() function that returns number[]</verify>
  <done>All 12 strategies implemented with test coverage for threshold generation</done>
</task>
